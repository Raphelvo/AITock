# AITock — Plateau Tock circulaire pour FoundryVTT

**Version : 1.1.0**

Ce module génère automatiquement un plateau de Tock **circulaire** pour FoundryVTT (compatible v12+), prêt pour **4 ou 6 joueurs**.

## Fonctionnalités

- Plateau circulaire généré sans image, purement avec les outils natifs de Foundry.
- Cases de parcours, de base, de maison pour chaque couleur.
- Jetons de départ générés, même sans joueur connecté.
- Prêt pour 4 ou 6 joueurs (modifier `TOCK_PLAYER_COUNT` dans `scripts/colors.js`).

## Installation/Mise à jour

1. **Supprimez l'ancienne version** du module dans `Data/modules/AITock` si besoin.
2. **Décompressez** la nouvelle archive ou copiez les nouveaux fichiers dans ce dossier.
3. **(Facultatif)** : Exécutez `install_AITock.bat` pour copier automatiquement le module dans le bon dossier.
4. Redémarrez Foundry, puis activez le module dans votre monde.
5. Le plateau se génère automatiquement à chaque lancement de la partie.
